# tiny-earth-dk-genomes


[TinyEarth Denmark](https://bce.au.dk/forskning/forsknings-og-udviklingsomraader/molecular-and-cellular-biotechnology-bio-og-kemiteknologi-technical-sciences-aarhus-universitet/research-groups-of-molecular-and-cellular-biotechnology/microbial-biosynthesis/projects/tiny-earth-danmark)
er en dansk gren af det internationale projekt ([tiny earth](https://tinyearth.wisc.edu/)). 
I et samarbejde mellem AU og AAU støttet af Novo Nordisk fonden er det blevet muligt at etablere genom sekventering som en udvidelse af projektet.
Bakteriernes genomer udvider betydeligt den viden man kan få om organismerne der er isoleret i projektet og deres evne til at producere kandidater til ny antibiotika.
